# ImageJ Macros Collection

연구에 필요한 ImageJ 매크로들을 정리해둔 공간입니다.

## 주요 기능
* **Multi-channel Plot:** 여러 채널의 line plot을 보기 좋게 그려줍니다.
* **Auto Scale:** y축 스케일을 자동으로 보정합니다.

[GitHub Repository 바로가기](https://github.com/MinLeeKR/ImageJ-Macros)

---

이 사이트는 ImageJ를 사용할 때 자주 쓰는 매크로들을 정리하고 공유하기 위한 공간입니다.  
각 매크로는 다음과 같은 형식으로 정리합니다.

- 어떤 문제를 해결하는지 (사용 시나리오)
- 입력 데이터 조건 (이미지 타입, 채널 수 등)
- 실행 방법 (설치 및 실행 순서)
- 결과 예시 (스크린샷)
- 매크로 코드 다운로드 링크

현재는 multi-channel line profile 매크로 두 가지를 예시로 정리해 두었습니다.
추가 매크로를 만들 때는 이 구조를 복사해서 새 페이지를 만들면 됩니다.
